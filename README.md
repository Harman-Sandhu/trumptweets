# faketrumptweets
Fake Trump Tweets API

## Usage/How to use

Send a GET request to ```https://faketrumptweets.herokuapp.com/tweet?text=```+your text.

## Docs
Documentation for this simple api can be found here: https://faketrumptweets.herokuapp.com/
